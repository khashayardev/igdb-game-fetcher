# راهنمای مشارکت در پروژه

از اینکه برای بهبود این پروژه وقت می‌گذارید، سپاسگزاریم!

## چگونه می‌توانید کمک کنید؟

### گزارش باگ‌ها
اگر باگی پیدا کردید، لطفاً در بخش [Issues](https://github.com/khashayardev/igdb-complete-scraper/issues) گزارش دهید.

### پیشنهاد ویژگی‌های جدید
ایده‌های خود را در بخش Issues مطرح کنید.

### بهبود مستندات
اگر مستندات نیاز به بهبود دارد، خوشحال می‌شویم Pull Request شما را ببینیم.

## فرآیند مشارکت

1. مخزن را **Fork** کنید
2. یک **Branch** جدید بسازید (`git checkout -b feature/amazing-feature`)
3. تغییرات خود را **Commit** کنید (`git commit -m 'Add amazing feature'`)
4. **Push** کنید (`git push origin feature/amazing-feature`)
5. یک **Pull Request** باز کنید

## نکات کدنویسی

- از Python 3.11+ استفاده کنید
- کد را با PEP 8 مطابقت دهید
- برای توابع و کلاس‌ها docstring بنویسید
- تغییرات خود را قبل از ثبت تست کنید

با تشکر از مشارکت شما!
